# Entregable TalentoTech - Practica de Seminario de Programación
Este repo es una practica basica sobre HTML y CSS para montar una pagina estática.
Contiene cosas muy basicas en el html como: Headers, paragraphs, Navbar, links, img, listas ordenadas y desordenadas, tablas y formularios.
El CSS contiene modificaciones basicas.
